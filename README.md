# 🔍 Admin Panel Scanner + Brute Force

Автоматический инструмент для поиска скрытых админ-панелей и подбора паролей на веб-сайтах.

## 📋 Описание

Инструмент сканирует веб-сайт на наличие скрытых административных панелей, определяет форму входа и автоматически подбирает логин/пароль из встроенных словарей.

### Особенности:
- 🎯 Автоматическое определение реальных страниц (обход SPA-ловушек)
- 🔐 Определение формы входа (WordPress и обычные формы)
- 💪 Встроенный брутфорс с топ-100 паролями
- 🚀 Работает с любыми сайтами (WordPress, CMS, SPA, кастомные)
- 📊 Подробный отчет о найденных панелях

## ⚙️ Настройка

### Изменение списка паролей

Отредактируйте переменную `TOP_PASSWORDS` в скрипте:

```python
TOP_PASSWORDS = [
    'admin', 'password', '123456',  # Добавьте свои пароли
    'my_custom_password',           # Ваш пароль
    # ... 
]
```

### Изменение списка логинов

```python
TOP_LOGINS = [
    'admin', 'root', 'administrator',  # Добавьте свои логины
    'my_username',                      # Ваш логин
]
```

### Настройка количества попыток

```python
# В коде, в циклах брутфорса:
for username in TOP_LOGINS[:10]:    # 10 логинов
    for password in TOP_PASSWORDS[:30]:  # 30 паролей
```

## 🛠️ Как это работает

1. **Калибровка** - определяет, как выглядит "фейковая" страница (404 или SPA-заглушка)
2. **Сканирование** - перебирает пути из wordlist
3. **Фильтрация** - отсеивает фейковые страницы по размеру ответа
4. **Определение формы** - ищет поля ввода пароля
5. **Брутфорс** - подбирает логин/пароль из встроенных словарей
6. **Отчет** - показывает найденные панели

## 🔧 Устранение проблем

### Проблема: "Ничего не найдено"
**Решение:**
- Увеличьте количество паролей в `TOP_PASSWORDS`
- Добавьте специфичные для сайта пути в wordlist
- Проверьте, не использует ли сайт капчу или 2FA

### Проблема: "Форма не обнаружена"
**Решение:**
- Сайт может использовать AJAX-логин (без традиционной формы)
- Проверьте вручную через браузер (F12 → Network)

### Проблема: "Таймауты"
**Решение:**
- Уменьшите количество потоков (если используете многопоточность)
- Увеличьте timeout в коде: `timeout=10`

## ⚠️ Предупреждение

**Используйте этот инструмент ТОЛЬКО на сайтах, где у вас есть письменное разрешение на тестирование безопасности!**

Несанкционированное сканирование и подбор паролей может быть незаконным и караться по закону.

## 💡 Советы

1. **Для SPA сайтов** (React/Vue/Angular) - скрипт может не найти форму, так как она генерируется JavaScript. В этом случае используйте ручной анализ через браузер.

2. **Для WordPress** - скрипт автоматически определит и использует правильные поля (`log`/`pwd`).

3. **Увеличьте скорость** - добавьте многопоточность или используйте `ffuf` для больших wordlist.

4. **Добавьте свои пароли** - самые эффективные пароли часто зависят от конкретного сайта.
